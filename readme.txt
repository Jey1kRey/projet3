Projet  : API d'un système de recommandation de films

Les bases de données et fichiers annexes ne sont pas intégrées aux scripts de codes ( poids fichiers)

premier_script_recom : programme pour créer les films avec les meilleurs notes, via pondération de la base
script k_mean : utilisation du clustering pour le système de recommandation
scritp recom_arbre : utilisation de l'arbre hiérarchique ascendant
script_reco_kppv : utilisation de l'algorithme des k plus proches voisins
temp_reference : établissement d'une liste de références de films pour effectuer état de l'art des distances disponibles
script_flask : production de l'API avec le module flask de python

second_script_recom : amélioration du sytème : utilisation d'un compteur de mots
